# Knowledge File Template

Use this template for public Rakos Media Group knowledge files.

This template is designed for entity, framework, service, concept, industry, and playbook content.

Every file should help people, search engines, AI search systems, answer engines, and future agents better understand Rakos Media Group, its services, concepts, frameworks, industries, and areas of expertise.

Do not publish private prompts, client strategies, scoring models, internal workflows, audit rubrics, or proprietary implementation details.
